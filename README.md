# Thammandru Jaswanth Guptha — Portfolio

> **"Building Intelligent Solutions Through Data and Code."**

A personal portfolio website for **Thammandru Jaswanth Guptha**, an AI & Data Science student, Full Stack Developer, and aspiring Data Engineer based in Visakhapatnam, Andhra Pradesh, India.

🌐 **Live Site:** [jaswanth-portfolio-inky.vercel.app](https://jaswanth-portfolio-inky.vercel.app/)

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | React 18 + TypeScript |
| Build Tool | Vite |
| Styling | Tailwind CSS |
| Icons | Lucide React |
| Backend / DB | Supabase |
| Deployment | Vercel |

---

## Features

- **Animated Hero Section** — Typing effect cycling through roles, particle canvas background, and profile image
- **About** — Personal introduction and background
- **Education** — Academic timeline
- **Skills** — Technology and tools showcase
- **Projects** — Highlighted builds including:
  - *MediPredict AI* — Medical prediction using AI/ML (HTML, CSS, JavaScript)
  - *LeafSense AI* — Plant disease detection from leaf images using Machine Learning
- **Certifications** — Certification cards with modal viewer
- **Contact** — Contact form with email/phone/social links, integrated with Supabase
- **Extras** — Loading screen, back-to-top button, smooth scroll navigation, scroll-triggered animations

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/jaswanthguptha/jaswanth-portfolio.git
cd jaswanth-portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Available Scripts

```bash
npm run dev        # Start dev server
npm run build      # Production build
npm run preview    # Preview production build locally
npm run lint       # Run ESLint
npm run typecheck  # TypeScript type checking
```

## Deployment

The site is deployed on **Vercel**. To deploy your own fork:

1. Push to GitHub
2. Import the repo in [vercel.com](https://vercel.com)
3. Vercel auto-detects the Vite config — no extra setup needed
